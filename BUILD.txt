
How to compile Yuni:
====================

1) Install CMake and a Subversion client

2) Checkout the source code:
  $ svn co http://svn.shikami.org/yuni/trunk

3) Generate the makefiles / projects:
  $ cmake -G <your_compiler> -DMODULES=all .

  You can run "cmake --help" for a list of generators for the -G option

4) To compile, either:
  - Run make (for Makefile-oriented generators) or
  - Launch the project file and tell your IDE to build the project (for Visual Studio / XCode / CodeBlocks / ... generators)
